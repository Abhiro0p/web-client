Wichtige Informationen zum Kartenportal

smartdemography setzt zur Darstellung der Ergebnisse das GIS-gestützte, raum-zeitliche Monitoring-Portal "KomMonitor" ein.
Gegenwärtig befindet sich diese Anwendung in einem kontinuierlichen Entwicklungsstadium. Sollten Sie eine fehlerhafte Darstellung oder unerklärliche Anwendungsfehler erhalten, so nehmen Sie bitte Kontakt zum Entwickler-Team auf. Im Sinne eines Testbetriebs können Sie die aktuelle Anwendungsversion ausprobieren, um eigene Daten ergänzen und wir laden Sie dazu ein, Feedback, Anregungen sowie Ideen für fehlendene Funktionalitäten an uns weiterzureichen.
